1) Правила подключения стилей прямо на странице используется тег style

<style>
    Правила стилей 
</style>

    - Данный кусок кода CSS должен распологаться перед тэгом </head>.
2) Правила написания стилей 
    - Начинается все с того, кому мы должны присвоить стили?
        class
        id
        tag
        atribute 
    - Мы должны открыть и закрыть фигурную скобку
        { }
        Целостная картина = 
        class {

        }
        id {

        }
        tag {

        }
        atribute {

        }

3) Закрывающая фигурная скобка должна быть параллельна названию класса или же любого свойства.
